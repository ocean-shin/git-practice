Abstract : This project is to practice git.

## Vim modes

- Normal mode : press 'esc' on any mode
- Insert mode : press 'i' on normal mode
- Visual mode : press 'v' on normal mode
- Command mode : press ':' on normal mode

## Installation

'''shell
$ git clone {repo address}
$ cd {repo name}
$ vi readme.md
'''
